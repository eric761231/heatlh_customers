# 立即開始 - 使用 IntelliJ IDEA

## ✅ 確認狀態

- ✅ Java 21 已安裝
- ❌ Maven 未安裝

## 🎯 解決方案：使用 IntelliJ IDEA

**不需要安裝 Maven！** IntelliJ 會自動處理所有依賴。

## 📥 下載安裝（5 分鐘）

1. **下載 IntelliJ IDEA Community**
   - 前往：https://www.jetbrains.com/idea/download/
   - 點擊 "Download" 下載 Community 版本（免費）
   - 檔案大小約 600MB

2. **安裝**
   - 執行下載的安裝檔
   - 按照安裝精靈完成安裝
   - 安裝完成後啟動 IntelliJ IDEA

3. **開啟專案**
   - 啟動 IntelliJ IDEA
   - 點擊 "Open" 或 File > Open
   - 瀏覽到 `java-backend` 資料夾
   - 點擊 "OK"

4. **等待自動設定**
   - IntelliJ 會自動識別這是 Maven 專案
   - 右下角會顯示 "Importing Maven projects..."
   - 第一次可能需要 2-5 分鐘下載依賴
   - 等待完成

5. **啟動應用程式**
   - 在左側專案視窗找到 `src/main/java/com/heath/api/Application.java`
   - 右鍵點擊該檔案
   - 選擇 "Run 'Application.main()'"
   - 或點擊右上角的綠色三角形 ▶️

6. **確認啟動成功**
   - 應該會看到：
     ```
     =========================================
     葡眾愛客戶管理系統 - 後端 API 已啟動
     API 端點: http://localhost:8080/api
     =========================================
     ```

## 🎉 完成！

現在後端已經在運行，可以：
1. 測試 API：http://localhost:8080/api/customers?userId=test-user-id
2. 開啟前端：`html/login.html`
3. 開始使用系統

## ⚠️ 如果不想安裝 IntelliJ

那麼需要安裝 Maven（見下方方案 2）。
