# Java 後端說明

## 目前狀態

Java 後端程式碼已經寫好了，但需要安裝 Maven 才能下載依賴。

## 如果未來需要使用 Java 後端

### 最簡單的方式：使用 IntelliJ IDEA

1. 下載 IntelliJ IDEA Community（免費）
2. 開啟 `java-backend` 資料夾
3. IntelliJ 會自動處理所有依賴
4. 不需要手動安裝 Maven

### 或者：使用線上平台

- Heroku、Railway、Render 等平台
- 他們會自動處理 Maven 和部署

## 目前建議

**繼續使用 Supabase**，因為：
- ✅ 已經運作正常
- ✅ 不需要額外設定
- ✅ 程式碼已經完成

如果未來需要更複雜的業務邏輯，再考慮使用 Java 後端。
